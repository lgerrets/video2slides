# Video to slides

## Install

> pip install -r requirements.txt

## Run base app

> python video2slides.py

## Run GUI (recommended)

> python gui.py

To build the executable:

> pip install pyinstaller
> pyinstaller gui.py

## Run web app (bugged)

> export FLASK_APP=app.py
> flask run

Then visit 
> http://127.0.0.1:5000/
